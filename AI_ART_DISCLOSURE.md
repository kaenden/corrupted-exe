# CORRUPTED.EXE — AI Content Disclosure

_Prepared for platform submission (CrazyGames / Poki). Provide on request._

## Summary
**Most of CORRUPTED.EXE's visuals are procedural** — drawn in code at runtime (the player, platforms,
spikes, UI panels, neon glow, particles, grid, vignette, star/shard/key/bug icons are all generated by
`BootScene.createPlaceholders()`). **All audio is procedural** — music and SFX are synthesized live with
the Web Audio API (oscillators + envelopes); the game ships **zero audio files**.

A small set of **raster images** (`public/assets/images/*.png`) was **generated locally with AI** and
hand‑integrated. Of these, only `bg_menu.png` and `icon_shard.png` are loaded at runtime; the rest are
shipped fallbacks of the procedurally‑drawn equivalents.

## Tools & process (raster images only)
- **Generator:** **InvokeAI** (local install, HTTP API at `localhost:9090`), driven headlessly by
  `tools/corrupted_gen.py`.
- **Model:** **FLUX.2 Klein 4B (fp8)** — `flux-2-klein-4b-fp8` (base `flux2`, Black Forest Labs), with the
  flux2 VAE and a quantized Qwen3 text encoder. Sampler `euler`, 8 steps, CFG 1.0, txt2img, random seed.
  Exact model keys/hashes are pinned at `tools/corrupted_gen.py:20‑23` (verified against the installed API).
- **Pipeline:** prompt → generate → `tools/corrupted_process.py` (rembg background removal → auto‑crop →
  Lanczos resize to the exact in‑game texture size) → `public/assets/images/`. Raw 1:1 renders are kept in
  `tools/art-gen/raw/`. No manual repainting.
- **Output license:** FLUX outputs are usable commercially; Black Forest Labs does not claim ownership of
  generated outputs. All generation was local; assets are embedded in the game.

## Prompt integrity (no third‑party IP)
Prompts are **generic genre descriptions only** — geometric/neon "corrupted‑simulation" aesthetic. **No
brand, franchise, studio, game, or character name appears in any prompt or filename.** The player is a
generic "small sleek angular robot with a single glowing cyan visor slit" — no recognizable mascot, logo,
or emblem. Full per‑asset prompts are verbatim in `tools/corrupted_gen.py` (`SUBJECTS`, lines 58‑87) and
the shared style suffixes (lines 28‑54); available on request.

rendering, and procedural audio synthesis were authored by the developer. The display font is Google
**"Chakra Petch"** (OFL), self‑hosted in `public/assets/fonts/`.
